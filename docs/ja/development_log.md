# 開発ログ

## Phase 4 - Python TCP クライアント

### 概要

Windows側から通信確認できる Python 製 CLI クライアントを追加しました。

### 追加内容

- `client/python/tcp_client.py`。
- `--host` と `--port` のコマンドライン引数。
- 既存TCPサーバプロトコルに対応した対話入力。
- サーバ応答の標準出力表示。
- `Ctrl+C` での安全な終了処理。
- 分かりやすい接続エラーメッセージ。
- README と実装メモの更新。

### メモ

- クライアントはPython標準ライブラリのみを使用します。
- このフェーズではGUIは追加していません。
- TCPサーバ側のプロトコルは変更していません。

## Phase 3 - 基本 TCP サーバ

### 概要

C 言語で最初に動作するシングルクライアント TCP サーバを実装しました。

### 追加内容

- TCP ポート `5000` で待ち受ける基本 TCP サーバ。
- `PING`、`GET_STATUS`、`START`、`STOP`、`RESET`、`QUIT` の行単位コマンド処理。
- `STOP` と `RUN` の内部状態管理。
- 起動、接続、受信、送信、終了を標準出力へ表示するログ。
- ビルド、実行、手動テスト手順。
- 実装メモ。
- `CHANGELOG.md` と `CONTRIBUTING.md`。

### メモ

- 複数クライアント対応と GUI クライアントは今後のフェーズで扱います。
- CSV ログ、設定ファイル、CAN 連携、テストは Phase 3 の対象外です。
